\section{Introduction}

TeX explains how to typeset a book about TeX~\cite{knuth1984texbook}, while
LaTeX explains how to avoid thinking about many details until one detail goes
missing~\cite{lamport1994latex}. This paper asks a smaller question: can the
default remote compiler receive several ordinary files and return one ordinary
PDF?

The expected answer is yes. Writing down the expectation does not make the
experiment redundant; it merely gives the failure something polite to
disagree with.
