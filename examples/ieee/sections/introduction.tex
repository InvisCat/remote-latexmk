\section{Introduction}

TeX was designed to produce high-quality typesetting~\cite{knuth1984texbook},
and LaTeX made structured documents pleasantly routine~\cite{lamport1994latex}.
IEEE provides conference templates in LaTeX~\cite{ieeeAuthorCenter}. This paper
uses that familiar structure to investigate a question that has troubled no
one for almost several minutes:

\begin{quote}
If a remote compiler successfully returns a PDF, has a PDF been returned?
\end{quote}

The question is deliberately circular. The build is not. A useful build must
fail when an input section, figure, bibliography, class, or compiler is
missing. This gives the circular question a surprisingly non-circular method.
